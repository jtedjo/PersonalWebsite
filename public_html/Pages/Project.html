Process Resource Simulator

Keywords: Process, Resource management, C++

This is a program written for Linux/Unix formatted text files that simulates Process and Resource Management. The program takes into account three different states, ready, running and blocked. The process have different operations such as create, request, release, and destroy. This aspect is simulated by a text file input. There are 4 resources, having {{1},{2},{3},{4}} resources. When process request resources, it checks whether or not the resource is available, if not then it will become blocked. The process also have a priority assigned, and there is only a single process active (a single core machine). The process also have a "parent" process and potentially a "child" process, with the topmost being the "init". Destroying a process also destroys its children and releases resources. The process will be unblocked based on FIFO order of a queue of the particular resource they are blocked on.

Search Engine Tokenizer

Keywords: Python, TF-IDF, NLTK, MySQL, Tokenizer, regex

A class project where given the contents of a web crawler( i.e. hundreds of HTML files). In our case we were given 37,500 documents with 164,874 unique tokens totalling 471,000 kb of index.

we converted the page contents into tokens which is then put into a MySQL database for processing. In specific, we used the Python NLTK lib for tokenizing, and the MySQL connector to populate the database. We then use MySQL statements to determine the TF-IDF scores for every term/document pair.

The user can search for particular keyword and the program will give the first 100 pages based on ranking of the TF-IDF.

Health Fitness Tracker

2nd place at MiraCosta College Programming Competition.

Keywords: Java, fitness, nutrition, biochemistry, GUI

Health Fitness Tracker(HFT) is the culmination of my interest in fitness/biology as well as technology. The program is written in Java. The program takes in users information such as their height, weight, age, gender, and fitness level to give diet recommendation, calculate BMI, as well as track caloric expenditure.

This program involved researching through various scientific papers on biochemistry and nutritions to implement the correct formulas for individualized diet. The program will determine based on your personal information the protein/fat/ and carbohydrate break down you should consume. The program has an interactive GUI, logging functionality, and login capabilities so more than one user can use the program.
